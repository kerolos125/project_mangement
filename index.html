Project Management - Git Workflow

This project demonstrates how a team collaborates using Git & GitHub.

We followed standard Git practices to simulate real-world version control workflows.

Technologies Used

  • Git
  • GitHub
  • VS Code

Key Features

  • Branch Management
  • Stashing and Merging
  • Reset and Undo

Git Commands Summary

Basic Git Commands

git init


git status


git add .


git commit -m "message"


git checkout -b feature/branch-name


git merge feature/branch-name

Remote & Advanced Commands

git remote add origin 
https://github.com/your-username/project.git


git push -u origin master


git pull origin master


git stash


git stash pop


git reset --soft HEAD~1


git reset --hard HEAD~1

Workflow Summary

  • Created a new Git branch for the contact form feature.
  • Stashed changes, made new commits, then restored the stash.
  • Pushed changes to GitHub and pulled updates.
  • Practiced undoing commits using reset.
